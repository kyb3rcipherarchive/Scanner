# CTF Scanner

## Scanner for CTFs

Install:
```bash
git clone https://github.com/kyb3rvizsla/Scanner.git
cd CTF-Scanner
sudo make install
cd ..
rm -rf CTF-Scanner
```

Unistall:
```bash
git clone https://github.com/kyb3rvizsla/Scanner.git
cd CTF-Scanner
sudo make unistall
```

## Use: 

```console
scanner <Mode> <IP>
```

Modes:

* Fast (**-f**)
* Detailed (**-d**)
* Aggresive (**-a**)
